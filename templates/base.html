Face Recognition Application

Gender Classification: This app is driven by data and machine learning model. When a user uploads an image, the app processes it in the background by:

  • Converting the image to grayscale
  • Detecting and cropping the face
  • Transforming it into an eigenimage
  • Using the model to make a prediction

All these functionalities are integrated into the Flask application
